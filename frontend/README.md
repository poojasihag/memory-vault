# 🎨 Memory Vault — Frontend

This is the frontend application for **Memory Vault**, built with React, Vite, and TypeScript.

## 🚀 Development

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

The app will start at `http://localhost:5173`.

### Environment Variables

The frontend connects to the backend at `http://localhost:8000/api`. Ensure the backend is running. (If you change the backend port, update the API base URL in `src/api/auth.ts` or similar core API files).

## 🛠️ Main Technologies

- **React + Vite** (Fast HMR)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Zustand** (State Management)
- **Lucide React** (Icons)
- **Konva** (Image Editing Context)
